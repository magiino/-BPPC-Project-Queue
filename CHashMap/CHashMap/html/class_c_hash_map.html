CHashMap::CForwardIterator & CHashMap::Erase ( CHashMap::CForwardIterator &  aIter)

Erase pair pointed by CForwardIterator.

Erase pair of the CHashMap which is pointing by CForwardIterator. aIter must point to the pairs in the same map.

Parameters
[in,out]aIterIterator pointing to the pair value in the CHashMap. Method update aIter after removing Pair to the new valid value.
Returns
New valid value of iterator. Iterator now pointing to the pair after removed pair if there exist.
Attention
Method generate std::runtime_error exception if aIter is invalid iterator (nullptr)
Method generate std::logic_error exception if aIter pointing to the different HashMap than is associated with this
